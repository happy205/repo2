# 잠실 장미상가 프라이빗 예약제 카페 웹사이트

GitHub Pages에 배포 가능한 정적 웹사이트입니다.

## 반영된 요구사항
- 다양한 원산지의 드립 커피 원두 제공 안내
- 1인당 컵 1개 기준, 예약 시간 동안 **커피 2회 + 물 2회 제공** 정책 반영
- 시간 단위 예약 + 예상 요금 계산
- 요금: **1시간 기준 1인 5,000원** (관리자 페이지에서 변경 가능)
- 무료 Wi-Fi 제공 안내 (관리자 페이지에서 변경 가능)
- 좌석 제약: **4인석은 3인 이상**, **6인석은 4인 이상** (관리자 페이지에서 변경 가능)
- 관리자 페이지에서 예약 현황 조회 + 영업시간/운영정책 설정 + 로그인 인증

## 페이지 구성
- `index.html`: 메인/예약 페이지
- `beans.html`: 원두 소개 페이지
- `guide.html`: 이용 가이드 페이지
- `admin.html`: 관리자 페이지(로그인 필요)

## 관리자 로그인
- URL: `/admin.html`
- 초기 계정: `admin`
- 초기 비밀번호: `1234`

> 현재 구현은 정적 사이트 환경을 고려한 클라이언트 인증(세션스토리지)입니다.
> 운영 환경에서는 Firebase Auth/서버 인증으로 교체를 권장합니다.

## 관리자에서 가능한 설정
- 요금(시간당 1인)
- 음료 제공 정책 문구
- Wi-Fi 제공 여부
- 4인석/6인석 최소 인원
- 요일별 영업시간(휴무 포함)
- 예약 현황 테이블 확인(로컬/Firebase 데이터)

설정은 `localStorage(jamsil-admin-settings)`에 저장되며, 메인 페이지 새로고침 시 반영됩니다.

## 설정(config.js)
`config.js`는 Firebase 연결 및 기본 영업시간을 정의합니다.

## 로컬 실행
```bash
python3 -m http.server 4173
```
브라우저에서 `http://localhost:4173` 접속
